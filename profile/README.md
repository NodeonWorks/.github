# NodeonWorks

Building software for businesses that move things — orders, inventory, logistics.

Our flagship product is **[Nodeon](https://nodeon.it)**, a multi-tenant B2B ERP platform for order management, warehouse operations, and supply chain.

## Products

| Repo | Description |
|------|-------------|
| [nodeon-core](https://github.com/NodeonWorks/nodeon-core) | Nodeon ERP platform — backend, frontend, infrastructure |
| [nodeon-site](https://github.com/NodeonWorks/nodeon-site) | Public marketing site (nodeon.it) |
| [nodeon-einvoicing](https://github.com/NodeonWorks/nodeon-einvoicing) | E-invoicing module (fatturazione elettronica SDI) |

## Stack

TypeScript · React · Node.js · PostgreSQL · AWS CDK
